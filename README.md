# Description
This website is an e-commerce website that sells artworks posted by artists.

# Function
* Register/login as a regular user or artist
* Address (CRUD)
* Cart (CRUD)
* Wishlist (CRUD)
* Payment by PayPal
* Artist gallery (CRUD)
* Artwork (CRUD)
* Search Artists and Artworks

# This is created by:
* HTML
* CSS
* JavaScript
* Bootstrap
* SCSS
* ASP.NET
* Paypal Sandbox API
* MySql
